Generation Prompt

Copy this entire prompt and send it to any AI chatbot along with your topic.

Generate a quiz in the following strict JSON format. Output ONLY the JSON array, no other text, no markdown code fences, no explanation. The topic/material will be provided by the user after this prompt.

Rules:
- Generate 10-20 questions total
- Mix both multiple choice (MCQ) and true/false (TF) questions
- Aim for roughly 60-70% MCQ and 30-40% True/False
- Each question must have a "type" field: either "mcq" or "tf"
- MCQ questions must have exactly 4 options (A, B, C, D) and answer must be A, B, C, or D
- TF questions must have exactly 2 options: {"A": "True", "B": "False"} and answer must be A or B
- Only one correct answer per question
- Include a brief explanation for the correct answer
- Questions should vary in difficulty
- Make questions clear and unambiguous

Required JSON format (output ONLY this array):

[
  {
    "type": "mcq",
    "q": "What is the question text?",
    "options": {
      "A": "First option",
      "B": "Second option",
      "C": "Third option",
      "D": "Fourth option"
    },
    "answer": "A",
    "explanation": "Brief explanation of why A is correct."
  },
  {
    "type": "tf",
    "q": "The Earth revolves around the Sun.",
    "options": {
      "A": "True",
      "B": "False"
    },
    "answer": "A",
    "explanation": "The Earth orbits the Sun in an elliptical path."
  }
]
