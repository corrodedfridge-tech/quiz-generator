Generation Prompt

Copy this entire prompt and send it to any AI along with your topic.

Generate a quiz in the following strict JSON format.
Output ONLY the JSON array — no other text, no markdown code fences, no explanation before or after.

IMPORTANT RULES:
- Focus on the MOST IMPORTANT concepts, key facts, critical definitions, and essential knowledge of the topic
- Prioritize questions that test understanding of core principles, not trivia
- Generate 15-25 questions total
- Mix FOUR question types: multiple choice ("mcq"), multi-select ("multi"), true/false ("tf"), and short answer ("short")
- Aim for roughly 40% MCQ, 20% Multi-Select, 20% True/False, 20% Short Answer
- Questions should range from foundational to challenging
- Make questions clear, precise, and unambiguous

FORMAT FOR EACH TYPE:

MCQ — must have exactly 4 options (A, B, C, D), answer is one letter:
{
  "type": "mcq",
  "q": "Question text?",
  "options": {"A": "Option 1", "B": "Option 2", "C": "Option 3", "D": "Option 4"},
  "answer": "B",
  "explanation": "Why B is correct."
}

MULTI-SELECT — must have 4-6 options, answer is array of correct letters:
{
  "type": "multi",
  "q": "Select ALL that apply: question text?",
  "options": {"A": "Option 1", "B": "Option 2", "C": "Option 3", "D": "Option 4", "E": "Option 5"},
  "answer": ["A", "C", "E"],
  "explanation": "Why A, C, and E are correct."
}

TRUE/FALSE — options must be exactly {"A": "True", "B": "False"}, answer is A or B:
{
  "type": "tf",
  "q": "Statement to evaluate.",
  "options": {"A": "True", "B": "False"},
  "answer": "A",
  "explanation": "Why it is true."
}

SHORT ANSWER — provide the correct answer and a list of accepted alternative answers:
{
  "type": "short",
  "q": "Question requiring a short text answer?",
  "answer": "Mitochondria",
  "accept": ["mitochondria", "the mitochondria", "mitochondrion"],
  "explanation": "The mitochondria is the powerhouse of the cell."
}

Output ONLY the JSON array. Now generate a quiz focusing on the most important concepts of:

💡 Add your topic at the end. Example: "...important concepts of: Calculus Derivatives"
