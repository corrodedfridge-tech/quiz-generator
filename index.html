Generation Prompt
Works with ChatGPT, Claude, Gemini, Llama — any LLM.
Generate a quiz in the following strict JSON format.
Output ONLY the JSON array — no other text, no markdown code fences.

RULES:
- Focus on the MOST IMPORTANT concepts, key facts, and essential knowledge
- Generate 20-30 questions total
- Mix SEVEN question types with roughly this distribution:
  30% MCQ, 15% Multi-Select, 15% True/False, 10% Short Answer, 10% Matching, 10% Ordering, 10% Fill-in-the-Blank
- Range from foundational to challenging
- Make questions clear, precise, and unambiguous

FORMAT FOR EACH TYPE:

MCQ — exactly 4 options (A-D), answer is one letter:
{"type":"mcq","q":"Question?","options":{"A":"Opt 1","B":"Opt 2","C":"Opt 3","D":"Opt 4"},"answer":"B","explanation":"Why B is correct."}

MULTI-SELECT — 4-6 options, answer is array of ALL correct letters:
{"type":"multi","q":"Select ALL that apply:","options":{"A":"Opt 1","B":"Opt 2","C":"Opt 3","D":"Opt 4","E":"Opt 5"},"answer":["A","C","E"],"explanation":"Why A, C, E."}

TRUE/FALSE — options must be {"A":"True","B":"False"}:
{"type":"tf","q":"Statement.","options":{"A":"True","B":"False"},"answer":"A","explanation":"Why true."}

SHORT ANSWER — provide answer and accepted alternatives:
{"type":"short","q":"Question?","answer":"Mitochondria","accept":["mitochondria","the mitochondria"],"explanation":"Explanation."}

MATCHING — 4-5 pairs, terms and definitions:
{"type":"match","q":"Match each term with its definition:","pairs":[{"term":"DNA","definition":"Genetic material"},{"term":"RNA","definition":"Messenger molecule"},{"term":"ATP","definition":"Energy currency"},{"term":"NADH","definition":"Electron carrier"}],"explanation":"Explanation."}

ORDERING — 4-6 items to arrange in correct order:
{"type":"order","q":"Arrange in chronological order:","items":["First event","Second event","Third event","Fourth event"],"explanation":"Explanation."}

FILL-IN-THE-BLANK — sentence with ___:
{"type":"fill","q":"The ___ is the powerhouse of the cell.","answer":"mitochondria","accept":["mitochondria","mitochondrion"],"explanation":"Explanation."}
